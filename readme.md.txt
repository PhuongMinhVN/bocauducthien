1. Nội dung chi tiết cho Website "Bồ Câu Đức Thiện"
Tiêu đề (Hero Section):
Slogan: Trang Trại Bồ Câu Đức Thiện – Đặc Sản Sạch Từ Đất Ninh Bình.

Giới thiệu ngắn: Chuyên cung cấp bồ câu ra ràng, bồ câu giống chất lượng cao tại Ninh Sơn - Ninh Bình. Giao hàng tận nơi, giá gốc tại trang trại.

Chuyên mục sản phẩm:
Bồ câu ra ràng (Thịt): Chim từ 20-25 ngày tuổi, thịt dày, thơm, giàu dinh dưỡng. Phù hợp cho nhà hàng, tiệc cưới, bồi bổ sức khỏe.

Bồ câu giống: Giống bồ câu Pháp, bồ câu ta chọn lọc. Khỏe mạnh, dễ nuôi, hỗ trợ kỹ thuật làm chuồng trại cho bà con.

Thế mạnh của trang trại (Tại sao chọn chúng tôi?):
Nguồn gốc rõ ràng: Nuôi tại Ninh Sơn, Ninh Phúc, Ninh Bình.

Chăn nuôi tự nhiên: Thức ăn sạch, không chất tăng trọng.

Giá cả cạnh tranh: Đổ buôn trực tiếp, không qua trung gian.

Phục vụ tận tâm: Ship nhanh nội tỉnh và các tỉnh lân cận.

Thông tin liên hệ (Footer):
Địa chỉ: Ninh Sơn, TP. Ninh Bình (gần khu vực Ninh Phúc).

Hotline/Zalo: [Ông điền số chủ trại vào đây]

Email: contact@bocauducthien.com

2. Prompt để Agent tạo Website (Copy đoạn này)
Ông copy đoạn dưới đây dán vào mấy con Agent chuyên code web nhé:

Role: You are an expert Web Developer and UI/UX Designer.
Task: Create a modern, professional, and mobile-responsive single-page (landing page) website for a pigeon farm named "Bồ Câu Đức Thiện" located in Ninh Bình, Vietnam.

Technical Requirements:

Use HTML5, Tailwind CSS, and FontAwesome for icons.

Modern, clean design with a focus on agriculture (Colors: Emerald Green, Deep Earthy Brown, and White).

Smooth scrolling and animations.

Sections to include:

Navigation: Logo (Bồ Câu Đức Thiện), Home, Products, About Us, Contact.

Hero Section: High-quality background image placeholder, catchy slogan "Bồ Câu Đức Thiện – Đặc Sản Sạch Từ Đất Ninh Bình", and a "Call Now" CTA button.

Products: 2 Cards (Pigeon Meat and Pigeon Breeding) with descriptions and price placeholders.

Features: 4 icons showing: Clean Feed, Natural Environment, Best Price, Fast Delivery.

Gallery: A grid for farm photos (placeholders).

Contact & Footer: Address in Ninh Sơn, Ninh Bình, Zalo button, and Google Maps embed placeholder.
Tone: Trustworthy, local, and professional. Please write all content in Vietnamese based on the info provided. Return the full HTML and CSS code in one block.
